ManU Website besticht durch brilliantes Layout

Gerade bin ich zufällig auf die offizielle Website von Manchester United gestoßen und habe mich doch schwer gewundert. Warum ist auf diesem Screenshot zu sehen. Das Layout ist komplett zerschossen, allerdings nur in Firefox, im Internet Explorer wird die Seite richtig dargestellt. Da tun sich Parallelen zur deutschen Telekom auf, die anscheinend ebenfalls keinen Wert darauf legt, dass ihre Website auch mit Firefox benutzbar ist.

Oder ist das beabsichtigt, damit das Nike-Logo besser sichtbar ist?
