Die bestbezahlten Fußballer 2006 im Web 2.0

Die Auflistung der Top-Verdiener im Profifußball 2006 allein ist auf Dauer dröge. Um mal zu sehen, was diese Spieler so drauf haben, habe ich ein wenig programmiert. Herausgekommen ist eine Seite, auf der durch alle Videos gebrowst (krasses Wort) werden kann, die mit den Namen dieser Spieler getaggt (auch nicht besser) sind.

Wie es sich im so genannten Web 2.0 gehört, kommen die Videos über eine Web Service-API von einer externen Website, YouTube, und können ohne die Seite neu zu laden (AJAX aber nicht die aus Amsterdam) auf torlaune.de durchgeklickt und angesehen werden können.

Viel Spaß damit und ich werde versuchen, mich demnächst wieder verständlich auszudrücken.
